## Практикум 01 - Задачи

**Задача 1:** Да се напише програма, която чете от конзолата 2 цели числа и отпечатва тяхното произведение.

Вход:
```c++
2 5
```

Изход:
```c++
10
```


**Задача 2:** Да се напише програма, която чете 2 цели числа *m* и *k* от конзолата. Програмата да отпечата колко секунди се съдържат в *m* часа и *k* минути.

Вход:
```c++
3 15
```

Изход:
```c++
11700
```

Вход:
```c++
a b c d f z
```

Изход:
```c++
1
```


Изход:
